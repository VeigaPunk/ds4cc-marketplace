DS4CC Marketplace

Public Git marketplace with 14 developer plugins for Grok Build, Codex, Claude Code, and Copilot CLI. The OpenAI submission is a separate, reviewed, read-only DS4CC app-plus-skills package.

Grok Build

grok plugin marketplace add VeigaPunk/ds4cc-marketplace
grok plugin install "VeigaPunk/ds4cc-marketplace#marketplace/plugins/myagents" --trust

Or paste GROK_PASTE.md into a Grok chat.

Codex

codex plugin marketplace add VeigaPunk/ds4cc-marketplace
