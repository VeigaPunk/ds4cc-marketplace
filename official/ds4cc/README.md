# DS4CC official submission package

This isolated package accompanies the read-only DS4CC MCP app. Its skill guides use of `browse_ds4cc_marketplace` and interpretation of reviewed result metadata only.

The app does not install, execute, or modify software. Any install command in a result is optional copyable text and must be independently reviewed before a user chooses to run it.

The package intentionally contains no instructions for registering or listing a broader developer marketplace.

Reviewed package source: <https://github.com/VeigaPunk/ds4cc-marketplace/tree/main/official/ds4cc>
