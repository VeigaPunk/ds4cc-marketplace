# The Musketeer

Use the plugin README as the canonical setup and command reference. The `grok`
CLI drives Grok features that are available through its authenticated web UI.

Keep CDP bound to loopback and use the dedicated shared browser profile. Run
`./install.sh` from this plugin directory to install the adapter.
